An IP address, which stands for Internet Protocol address, is a unique string of numbers separated by periods or colons that identifies each computer using the Internet Protocol to communicate over a network.
A public IP address is an address that is globally unique and accessible over the internet.
A private IP address is an address used within a private network, such as a local area network (LAN) or a home network.
